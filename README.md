# ExpenseManager

Native iPhone expense management app built with Swift and SwiftUI.

## Status

Local-first SwiftUI project. The first screen is a fast quick expense entry flow with in-memory expenses.

## Target

- Platform: iOS
- IDE: Xcode
- Language: Swift
- UI: SwiftUI
- App target: `ExpenseManager`
- Storage: in-memory first

## Current Direction

- Native iPhone app only
- No web app
- No Python desktop app
- No React/Vite
- No backend, authentication, cloud sync, or external service dependency
