# Roadmap

## Milestone 1: App Foundation

- Create a local SwiftUI iOS project.
- Add a simple app shell.
- Decide on the first domain models.
- Build the quick expense entry screen.

## Milestone 2: Fast Expense Entry

- Replace placeholder category icons with real image assets.
- Refine iPhone layout and gestures in Xcode Simulator.
- Add basic validation tests for amount entry.

## Milestone 3: Persistence

- Add a local storage layer.
- Save expenses with category, amount, and date/time.

## Milestone 4: Review

- Add monthly expense list.
- Add category totals.
- Add simple spending summaries.
